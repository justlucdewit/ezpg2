<h2>
    the second version of ezpygame<br>
    it is a python module build ontop of pygame<br>
    it is used just like p5js or processing, but<br>
    with python syntax!
</h2>
<h6>
    features:
    <ul>
        <h6><b>basic control</b></h6>
        <li><a>start()</a></li>
        <li><a>createCanvas()</a></li>
        <li><a>width()</a></li>
        <li><a>height()</a></li>
        <li><a>update()</a></li>
        <li><a>setTitle()</a></li>
        <li><a>frameCount()</a></li>
        <li><a>frameRate()</a></li>
        <li><a>consoleClear()</a></li>
        <br>
        <h6><b>math</b></h6>
        <li><a>math()</a></li>
        <li><a>cos()</a></li>
        <li><a>sin()</a></li>
        <li><a>tan()</a></li>
        <li><a>pi</a></li>
        <li><a>mapping()</a></li>
        <li><a>random()</a></li>
        <br>
        <h6><b>vector</b></h6>
        <li><a>createVector()</a></li>
        <li><a>vector.x</a></li>
        <li><a>vector.y</a></li>
        <li><a>vector.toString()</a></li>
        <li><a>vector.set()</a></li>
        <li><a>vector.copy()</a></li>
        <br>
        <h6><b>input</b></h6>
        <li><a>mouseX()</a></li>
        <li><a>mouseY()</a></li>
    </ul>
